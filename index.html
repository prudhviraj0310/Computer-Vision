Existing BTP ITMS Infrastructure Gaps
✕
No Helmet / Seatbelt AI: Existing cameras only trace red lights. Rider compliance is not checked automatically.
✕
No Triple-Riding Model: Motorcycles carrying 3+ passengers are not flagged by standard ITMS scripts.
✕
No Offender Profiling: Prior violations are indexed statically. Repeating offenders are not tracked across stations.
✕
No Predictive Dispatch: Force deployment is scheduled manually rather than guided by ML hotspot analytics.
✕
No Risk-Weighting: High-density hazards are mixed with minor side-street illegal parking events in the timeline.
